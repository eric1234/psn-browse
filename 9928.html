DiRT Rally

DiRT Rally is a racing video game focused on rallying. Players compete in timed stage events on tarmac and off-road terrain in varying weather conditions. On release, the game features 17 cars, 36 stages from three real world locations, and asynchronous multiplayer. Rallycross and player versus player multiplayer modes are planned. Codemasters announced a partnership with the FIA World Rallycross Championship in July 2015. The early access version contains cars from the 1960's, 70's, 80's, Group B, Group A, 2010s Modern Rally, and Pikes Peak.

About

A site for browsing the games available on PlayStation Now.

Previous options:

  • Official Site - Mostly text titles only. No search
  • App - Few broad category, poor search, limited game information
  • IGN - Decent category-based search. Poor other search options. Decent game info if a popular game.

The aim of this site is to be better. Titles are extracted from the official site on a daily basis so it is always up-to-date. Each title is then enriched with data for IGDB. Advanced faceted search to narrow down to the game you want. Open source so anybody can improve it.
